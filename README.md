## Operation

### Development

```console
$ cd snap-screener-form
$ yarn install
$ yarn start
```

### Build

```console
$ cd snap-screener-form
$ yarn install
$ yarn build
```

#### Local Serve

```console
$ PUBLIC=snap-screener-form/build bin/serve
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```
